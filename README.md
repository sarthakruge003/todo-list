# 📝 To-Do List Web App

A simple and responsive To-Do List web application built using **HTML**, **CSS**, and **JavaScript**. It allows users to:

- ✅ Add new tasks  
- ☑️ Mark tasks as completed  
- 🗑️ Delete tasks  
- 💾 Automatically save tasks using `localStorage`
